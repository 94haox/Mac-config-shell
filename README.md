# Mac-config-shell
Mac 配置脚本

### 使用说明

下载到文件夹，进入到对应目录

```
$ ./MacConfig.sh
```

### 更新说明 

2018.9.4
因为有些工具需要翻墙，所以在执行脚本之前，需要自备梯子，并且将终端配置好代理，让终端走代理，否则会出现安装失败
